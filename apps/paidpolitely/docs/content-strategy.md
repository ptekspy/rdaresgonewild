# PaidPolitely.com Content Strategy

PaidPolitely.com is a two-sided acquisition page.

## Audience 1: advertisers

Goal: get brands, platforms, services, and creator tools to buy direct display placements.

Primary CTA:

```txt
Buy display ads
```

Core promise:

```txt
Reach focused NSFW Reddit-grown communities without relying on broad, low-quality adult ad traffic.
```

## Audience 2: subreddits/mods

Goal: get moderators or community owners to start a conversation about joining the network.

Primary CTA:

```txt
Bring a subreddit
```

Core promise:

```txt
Turn repeated subreddit behaviour into a useful companion site with rankings, profiles, galleries, challenges, stats, and sponsor inventory.
```

## Tone

Professional, adult-aware, independent, direct. Not corporate. Not sleazy.
