# Copy Guide

## Brand Voice

The voice should be:

- short
- cheeky
- confident
- teasing
- competitive
- community-first

It should not be:

- corporate
- clinical
- vulgar
- over-explaining

## Core Taglines

```txt
Track dares. Level up. Get wild.
```

```txt
Your dare progress, ranked.
```

```txt
Complete more. Rank higher. Get wild.
```

```txt
The unofficial dare tracker for r/daresgonewild.
```

## Hero Copy

```txt
Track playbook progress, compare creators, and find your next challenge.
```

```txt
Enter a Reddit username to see completed dares, progress, and the next dare waiting.
```

## CTA Labels

Primary:

- Pick my next dare
- View leaderboard
- Track my progress
- Find a challenge

Secondary:

- See top creators
- Browse playbook
- Learn how it works

## Leaderboard Copy

```txt
Who’s leading the dare board?
```

```txt
Compare playbook completions, community dares, and overall progress.
```

```txt
Climb the ranks one dare at a time.
```

## Dare Picker Copy

```txt
Enter a Reddit username and we’ll suggest the next uncompleted dare.
```

```txt

```

```txt

```

## Empty State Copy

```txt
No tracked dares yet.
```

```txt
Start with a username and build from there.
```

```txt

```

## Footer Disclaimer Copy

```txt
Unofficial fan site. Not affiliated with Reddit or r/daresgonewild.
```

```txt
Adult community resource. Users must follow all platform rules and local laws.
```

## Words to Use

- dare
- challenge
- playbook
- progress
- level
- rank
- creator
- community
- streak
- completed

## Words to Avoid in Core UI

Avoid overly explicit or crude wording in navigation, buttons, and general UI labels. Keep the adult edge in the brand personality, not in every line of copy.
