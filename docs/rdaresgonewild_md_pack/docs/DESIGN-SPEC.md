# r/DARES Gone Wild — Website Design Specification

## Brand Summary

r/DARES Gone Wild is a cheeky, gamified, adult-themed community brand built around dares, progression, and playful competition.

The visual identity should feel:

- bold
- playful
- mischievous
- sleek and modern
- adult, but not explicit or crude
- energetic and reward-driven

The approved identity balances gaming / leaderboard energy with flirty personality.

## Core Brand Personality

### Keywords

- Cheeky
- Confident
- Flirty
- Competitive
- Gamified
- Modern
- Punchy
- Neon

### Tone

The website should feel like a mix of:

- an achievement platform
- a creator leaderboard
- a playful adult community
- a dare progression system

Avoid anything that feels porn-site tacky, overly dark and grimy, corporate, sterile, or childish.

## Logo System

The logo consists of:

1. Flame mascot — a cheeky flame with a subtle feminine silhouette hint.
2. r/ prefix — a nod to the subreddit identity.
3. DARES — bold main wordmark.
4. GONE WILD — secondary sub-line, with WILD in a brush-style treatment.

### Mascot Direction

The mascot should always remain:

- playful
- teasing
- confident
- suggestive in silhouette only
- not explicit
- not realistic

## Logo Usage Rules

### Clear Space

Maintain clear space equal to at least the width of the mascot eye area around all sides of the full logo.

### Minimum Size

- Horizontal logo in navbar: minimum 32px high
- Preferred navbar height: 40–56px
- Flame icon favicon: use exported favicon sizes

### Do Not

- stretch the logo
- rotate the logo
- add bevels or heavy shadows
- change the mascot expression
- recolor the logo outside approved variants
- place the color logo on busy backgrounds without enough contrast

```css
--brand-gradient: linear-gradient(135deg, #F9047C 0%, #FF2C76 35%, #FF4B2B 68%, #FF7A00 100%);
```

## Typography Recommendations

### Display / Headlines

- `Sora`
- fallback: `Space Grotesk`, `Inter`, `sans-serif`

### UI / Body

- `Inter`
- fallback: `system-ui`, `sans-serif`

### Type Scale

- Hero H1: `48–64px`, weight `700–800`
- Section H2: `32–40px`, weight `700`
- Card title: `20–24px`, weight `700`
- Body text: `16–18px`, weight `400–500`
- Small metadata: `13–14px`, weight `500`

## UI Design Direction

The site should feel:

- dark mode first
- neon-accented
- clean and polished
- card-based
- slightly sporty / competitive
