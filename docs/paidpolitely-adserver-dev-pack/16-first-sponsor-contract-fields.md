# First Sponsor Contract Fields

This is not legal advice. It is a practical data checklist for manually sold ads.

For each private sponsor, collect:

```txt
Advertiser name
Contact name
Contact email
Advertiser website/profile URL
Campaign name
Start date
End date
Contract price
Currency
Included sites
Included placements
Creative image URL/file
Headline
Body text
CTA text
Target URL
Whether image is explicit
Whether target site is adult
Any special restrictions
Notes
```

Recommended first pricing products:

```txt
Single placement weekly sponsor
Single placement monthly sponsor
Site-wide footer sponsor
Homepage + leaderboard bundle
Network starter bundle
```

Example product:

```txt
NSFW Network Starter Sponsor
Duration: 30 days
Includes:
- rdaresgonewild homepage_top
- rdaresgonewild leaderboard_top
- rdaresgonewild profile_sidebar
- future rprogresspics feed_top if available
```

For MVP billing, keep payment/contract management outside the app. Store `contractValue`, `currency`, and notes only.
